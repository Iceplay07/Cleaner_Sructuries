# Cleaner_Structures

**Cleaner_Structures** — это консольная утилита на C++, которая рекурсивно сканирует директорию и формирует отчёт о файлах.

Программа предназначена для анализа файловой структуры проекта и умеет находить:

- пустые файлы;
- точные дубликаты;
- похожие файлы по содержимому;
- игнорировать бинарные и нецелевые файлы;
- выводить отчёт в терминал;
- сохранять отчёт в файл.

---

## Возможности

- Рекурсивный обход директории.
- Игнорирование бинарных файлов и файлов неподходящих расширений.
- Определение пустых файлов.
- Поиск точных дубликатов.
- Поиск похожих файлов по значимым строкам.
- Настраиваемый порог похожести.
- Вывод результата в `stdout` или файл.
- Автоматическое создание папки для отчёта.
- Поддержка флагов `-o`, `--output`, `-t`, `--threshold`, `-h`, `--help`.
